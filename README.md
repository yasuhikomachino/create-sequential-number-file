# create-sequential-number-file

Create a sequential number file by day.

## Setup

```
$ chmod +x ./create.sh 
```

## Example

```
$ ./create.sh
[SUCCESS] Create dist/2020-10-10-001.md
```

```
$ tree ./dist
./dist
├── 2020-10-09-001.md
├── 2020-10-09-002.md
├── 2020-10-09-003.md
└── 2020-10-10-001.md
```
